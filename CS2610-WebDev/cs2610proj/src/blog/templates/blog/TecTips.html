Git Command Cheat Sheet

Git Command Description Example
Init create a new local repository: git init <directory name>
Clone Create a working copy of a local repository:

For a remote server, use:		 git clone /path/to/repository

git clone username@host:/path/to/repository
Connect to Remote Repository If you haven't connected your local repository to a remote server, add the server to be able to push to it: git remote add origin <server>
Add add a file to index before commiting git add <filename>
Commit Commit changes to head (but not yet to the remote repository): git commit -m "Commit message"
Push Send changes to the master branch of your remote repository: git push origin master
Status List the files you've changed and those you still need to add or commit: git status
Checkout Create a new branch and switch to it:

Switch from one branch to another:		 git checkout -b <branch_name>

git checkout <branch_name>
Pull Fetch and merge changes on the remote server to your working directory: git pull <remote>
